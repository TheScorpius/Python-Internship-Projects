# Day 04 – Utility Calculator Package (Python)

## Project Overview
The Utility Calculator Package is a modular Python project designed to demonstrate
the use of **functions, modules, and packages**.  
It provides multiple real-world utilities such as arithmetic operations, financial
calculations, unit conversions, and health-related calculations like BMI.

This project follows proper Python package structure and clean code practices.

---

## Concepts Covered
- Python functions
- Modules and packages
- Code reusability
- Menu-driven programs
- Conditional statements
- Clean project structure

---

---

## Module Description

### basic.py
- Addition
- Subtraction
- Multiplication
- Division

### finance.py
- Simple Interest calculation
- EMI calculation

### conversion.py
- Celsius to Fahrenheit
- Kilometers to Miles

### health.py
- BMI calculation
- BMI category classification

### __init__.py
Marks the `utilities` folder as a Python package, allowing modular imports.

## How to Run the Project
1. Open terminal in the project folder
2. Run the command: python main.py
