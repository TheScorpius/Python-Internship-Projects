# Day 05 – Todo List Manager

## Project Overview
This project is a command-line Todo List Manager built using Python.
It allows users to add, view, update, and delete tasks using lists and tuples.

---

## Concepts Used
- Lists
- Tuples
- Loops
- Conditional statements

---

## Project Structure
- todo.py – Core todo management logic

---

## How to Run
1. Open terminal in this folder
2. Run:
   python todo.py

---

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

---

## Learning Outcomes
- Managing data using lists and tuples
- Implementing CRUD operations
- Building menu-driven programs
