# Day 06 – Contact Management System

## Project Overview
A contact management system built using Python dictionaries and sets.
It allows users to add, view, search, and delete contacts while ensuring
unique phone numbers.

---

## Concepts Used
- Dictionaries
- Sets
- Loops
- Conditional statements

---

## Project Structure
- contact_manager.py – Core contact management logic

---

## How to Run
python contact_manager.py

---

## Features
- Add new contacts
- View all contacts
- Search contacts by name
- Delete contacts
- Prevent duplicate phone numbers

---

## Learning Outcomes
- Using dictionaries for structured data
- Using sets for uniqueness
- Implementing CRUD operations
