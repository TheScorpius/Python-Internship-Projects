# Day 07 – File-Based Attendance Management System

## Project Overview
This project is a file-based attendance management system built using Python.
It allows users to mark and view student attendance with persistent storage.

---

## Concepts Used
- File handling (read/write)
- Text file storage
- Loops and conditional statements

---

## Project Structure
- attendance.py – Main attendance logic
- attendance.txt – Stores attendance records

---

## How to Run
python attendance.py

---

## Features
- Mark student attendance
- View all attendance records
- Persistent data storage using files

---

## Learning Outcomes
- Implementing file-based data storage
- Reading and writing structured text files
- Building menu-driven applications
