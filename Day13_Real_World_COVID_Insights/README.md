# Day 13 – Real World COVID-19 Dataset Insight Generator

## Project Overview
This project is a **real-world data analysis and insight generator** built using Python.  
It analyzes a **country-wise COVID-19 CSV dataset** and generates meaningful insights using **Pandas** and **Matplotlib**.

The project focuses on transforming raw data into **visual and statistical insights**, which is a key real-world data handling skill.

---

## Objectives
- Load and process a real-world CSV dataset
- Perform country-wise and global analysis
- Identify top affected countries
- Generate visual insights using graphs
- Demonstrate practical data analytics skills

---

## Dataset Information
- **Dataset Type:** Country-wise COVID-19 data
- **Format:** CSV
- **Source:** Kaggle
- **Key Columns Used:**
  - Country/Region
  - Confirmed
  - Deaths
  - Recovered
  - Active

---

## Features
- Dataset overview and summary statistics
- Top 10 most affected countries analysis
- Country-specific COVID-19 insights
- Bar chart visualization for global comparison
- Pie chart visualization for country-level case distribution

---

## Technologies Used
- Python
- Pandas
- Matplotlib
